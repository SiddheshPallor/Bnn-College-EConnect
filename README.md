
# Bnn College E-Connect

It is a mobile application developed for Android phones that was developed to serve as a one-stop shop for students of Bnn College Students to find all the information they need.

In order to create a user-friendly interface, the app uses Material UI (subject to improvement) that is meant to automatically attract the user to remain on the app for as long as possible.

It provides an overview about the campus to students, such as the University's history, departments, galleries, faculties, and libraries etc.



## Features

- Splash Screen
- Home
- Notices
- Faculty
- Gallery
- About
- E-Book
- Video Lecture
- Themes
- Website
- Share
